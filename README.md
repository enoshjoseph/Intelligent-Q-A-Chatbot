# 🤖 Intelligent Multimodal Q&A Chatbot

An AI-powered chatbot that allows users to interact with **PDFs, Images, and Videos** using advanced Retrieval-Augmented Generation (RAG).

---

## 🚀 Features

- 📄 Chat with PDFs
- 🖼️ Extract insights from Images (OCR)
- 🎥 Analyze Videos using AI
- 🔎 Semantic Search using Vector Database
- 🤖 Context-aware responses using RAG
- ⚡ FastAPI backend + React frontend

---

## 🧠 Tech Stack

### Backend
- FastAPI
- ChromaDB (Vector Database)
- Sentence Transformers
- Whisper (Audio Transcription)
- Tesseract OCR
- PyMuPDF / PDF2Image
- FFmpeg

### Frontend
- React (Vite)
- Tailwind CSS
